# Unemployment Analysis in India

## Objective

The objective of this project is to analyze unemployment trends across India and understand the impact of COVID-19 on employment patterns using Exploratory Data Analysis (EDA) and data visualization techniques.

## Dataset Source

Dataset obtained from Kaggle:

* Unemployment in India Dataset
* Source: Kaggle
* Link: https://www.kaggle.com/datasets/gokulrajkmv/unemployment-in-india

## Dataset Features

The dataset contains state-wise unemployment statistics across India including:

* Region
* Date
* Frequency
* Estimated Unemployment Rate (%)
* Estimated Employed Population
* Estimated Labour Participation Rate (%)
* Area (Urban/Rural)

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Missing Value Handling
4. Exploratory Data Analysis (EDA)
5. State-wise Unemployment Analysis
6. Time-Series Analysis
7. Urban vs Rural Comparison
8. Employment Analysis
9. Correlation Analysis
10. Insight Generation

## Visualizations

* Top 10 States by Average Unemployment Rate
* Average Unemployment Trend Over Time
* Urban vs Rural Unemployment Comparison
* Top 10 States by Employment
* Correlation Heatmap

## Key Findings

* Tripura recorded the highest average unemployment rate among the analyzed states.
* A sharp increase in unemployment was observed during the COVID-19 lockdown period.
* Urban unemployment was higher than rural unemployment.
* Employment and unemployment exhibited a weak inverse relationship.
* Labour participation rate showed minimal correlation with unemployment.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Conclusion

The analysis demonstrates how economic disruptions such as the COVID-19 pandemic significantly affected employment opportunities across India. Regional differences and urban-rural disparities were also observed, highlighting the complexity of unemployment trends.
